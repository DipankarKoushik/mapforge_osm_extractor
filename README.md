OSM Extractor

A full-stack geospatial web application to extract, visualize, and export OpenStreetMap data.

Features
Dual Search: Search by City Name or Coordinates.
Custom Selection: Draw Polygons, Circles, or Rectangles.
Smart Export: Download as Shapefile (GIS), GeoJSON, SVG, or PNG.
Base maps: Includes Satellite and Dark Mode layers.

Tech Stack
Backend: Python, FastAPI, OSMnx, GeoPandas.
Frontend: HTML, JavaScript, Leaflet.js.
